An analogy to describe the differences between HTML and CSS

An analogy to describe the differences between HTML and CSS are like the differences between a builder and a interior designer. The builders job is to lay the foundations of the huose and make the walls, ceiling, and roof. Once that is done the interior designer can come in and decorate the interior space. Like this example, HTML acts as the foundations to CSS as a form of styling the foundations of a website.
